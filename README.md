# FIFA 21 — Data Preprocessing Impact on Model Accuracy
### Data Analytics Module | Week 3 Group Assignment

## Analytical Statement
> **"Data Preprocessing Has a Significant Impact on Model Accuracy"**

This project investigates and statistically justifies the above statement using the FIFA 21 Messy Raw Dataset, applying Descriptive, Inferential, and Predictive Analytics.

- **Target Variable:** OVA (Overall Player Rating)

## What Each Script Does

### 01_preprocessing.R — Member 1
- Loads the raw FIFA 21 dataset
- Documents all data quality issues
- Removes duplicates
- Fixes currency columns (Value, Wage) — strips €/£ symbols, converts K/M
- Standardises Height and Weight to metric units
- Fixes star rating columns (★ → integer)
- Imputes missing values (median for numeric, mode for categorical)
- Encodes categorical columns (Label Encoding)
- Removes outliers using IQR method
- Scales numeric columns using StandardScaler
- Saves `data/fifa21_clean.csv`

### 02_descriptive.R — Member 2
- Loads both raw and clean datasets
- Computes summary statistics (Mean, Median, SD, Skewness, Kurtosis)
- Produces before vs after comparison table
- Generates charts:
  - Missing values bar chart
  - Histogram of Overall Rating (before vs after)
  - Box plots for Wage and Value
  - Correlation heatmaps (raw vs clean)
  - Top 10 player positions frequency chart
- Saves all charts to `outputs/charts/`
- Saves statistics table to `outputs/results/descriptive_stats.csv`

### 03_inferential.R — Member 3
- Loads raw and clean datasets (numeric columns only)
- States H₀ and H₁ formally
- Runs 10-fold cross validation on both datasets using Random Forest
- Extracts 10 R² scores from each
- Runs Paired t-test (`t.test` with `paired=TRUE`)
- Calculates Cohen's d (effect size)
- Calculates 95% Confidence Interval
- Saves fold scores comparison chart to `outputs/charts/`
- Saves full hypothesis results table to `outputs/results/hypothesis_results.csv`

### 04_predictive.R — Member 4
- Loads raw and clean datasets
- Splits data 80/20 (train/test) with fixed seed (42)
- Trains 3 models on BOTH raw and clean data:
  - Linear Regression
  - Decision Tree
  - Random Forest
- Evaluates each model: R², RMSE, MAE
- Builds full comparison table (Raw vs Preprocessed)
- Generates R² and RMSE bar charts
- Saves charts to `outputs/charts/`
- Saves comparison table to `outputs/results/model_comparison.csv`

---

## Hypothesis

| | Statement |
|--|-----------|
| **H₀** | Preprocessing has no significant effect on model accuracy |
| **H₁** | Preprocessing significantly improves model accuracy |
| **Significance Level** | α = 0.05 |
| **Test Used** | Paired t-test |

If **p-value < 0.05** → Reject H₀ → Preprocessing significantly improves accuracy ✅
